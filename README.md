# PitchUp – Sports Venue Marketplace

PitchUp is a SaaS + marketplace platform that allows sports venues
to manage facility bookings, availability, and payments.

## Problem

Sports venues often manage bookings across spreadsheets,
email, and multiple tools.

This leads to double bookings, poor availability visibility,
and manual payment management.

## Solution

PitchUp provides a unified platform for venue operators to manage:

• facility availability
• booking requests
• payments and payouts
• customer management
• venue dashboards

## Tech Stack

Frontend
React + TypeScript + Tailwind

Backend
FastAPI (Python)

Database
Supabase Postgres

Payments
Stripe Connect marketplace

## Architecture

React frontend
↓
FastAPI API layer
↓
Supabase (Postgres + Auth)
↓
Stripe Connect payouts

## Key Features

• venue onboarding workflow  
• facility management dashboards  
• slot-based booking engine  
• calendar availability system  
• role-based access control  
• marketplace payments
